
# Fetch Data using Axios & React Query

## Overview 
This is demo for fetching data using `Axios` & `React Query` in React Application. 

## Demo
Fork and run this application using below command:
```
npm install
npm run dev
```
## Tech Stack

* HTML5
* CSS3
* TypeScript 
* React.js 18
* Tailwind CSS
* Axios
* React Query
